GENERATED BLOCK RESOURCE ATLAS

Every generated block is a public resource rule.

NiceChunk maps terrain blocks into scientifically scoped resource foundations. Each block exposes its generation signal, likely elemental composition range, and role in the future mining and forging economy.

Raw resources are mined from generated blocks. They must be smelted into material resources before forging.

SMELTING DATA

Raw resources become forgeable materials.

Raw resources cannot be forged directly. A smelter consumes fuel heat, checks the material recipe, then outputs material resources for item forging.
